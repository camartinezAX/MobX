Key Discovery: Shared Locators!

OutSystems generates identical element IDs for both web and mobile builds. This means we can maintain one locator file that works across both platforms.

‚ö†Ô∏è Important: While locators are shared, test scripts need separate implementations due to platform differences (browser vs. app setup, mobile-only features like biometrics, and different user journeys).
